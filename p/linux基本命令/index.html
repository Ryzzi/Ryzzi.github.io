基础命令

一、目录与文件操作

1. 目录切换查看 

1
2
3
4
5
6
7
8

pwd                 # 查看当前所在路径
ls                  # 列出当前目录文件
ls -l               # 详细列表（权限、大小、时间）
ls -a               # 显示隐藏文件（.开头）
cd /xxx             # 进入指定目录
cd ..               # 返回上一级
cd ~                # 回到家目录
cd -                # 回到上一次所在目录

2. 新建 / 删除 / 移动 / 复制 

1
2
3
4
5
6
7
8

mkdir test          # 创建文件夹test
mkdir -p a/b/c      # 递归创建多级目录
rm file.txt         # 删除文件
rm -rf test/        # 强制删除文件夹（慎用，无回收站）
cp a.txt /tmp/      # 复制文件
cp -r dir /tmp/     # 复制文件夹
mv old.txt new.txt  # 重命名文件
mv file /tmp/       # 移动文件

3. 文件读写查看 

1
2
3
4
5
6

cat test.txt        # 一次性打印全部内容
less test.txt       # 分页查看（空格翻页 q退出）
head -10 test.txt   # 看前10行
tail -20 test.txt   # 看最后20行
tail -f test.log    # 实时跟踪日志（排查程序输出）
touch a.txt         # 创建空文件/更新文件时间

二、文本搜索处理 

1
2
3
4

grep "关键词" test.txt    # 文件内搜索关键词
grep -r "key" ./          # 递归搜索整个目录所有文件
find ./ -name "*.log"     # 查找后缀log的文件
sed / awk                 # 高级文本替换、统计（脚本常用）

三、权限 & 用户操作

1. 权限修改 

1
2

chmod 755 run.sh      # 修改文件读写执行权限
chown user:group file # 修改文件归属用户/组

2. 用户切换 

1
2
3
4
5

whoami                # 查看当前登录用户
su root               # 切换root用户
sudo 命令             # 临时用管理员权限执行单条命令
useradd testuser      # 新建用户
passwd testuser       # 设置用户密码

四、进程、系统资源查看 

1
2
3
4
5
6
7

ps -ef                # 查看所有进程
top / htop            # 实时查看CPU、内存、进程占用
kill 1234             # 终止PID=1234进程
kill -9 1234          # 强制杀死卡死进程
df -h                 # 查看磁盘分区占用
free -h               # 查看内存使用
uptime                # 查看开机时长、负载

1
2
3
4
5

ip addr / ifconfig    # 查看本机网卡IP
ping www.baidu.com    # 测试网络连通
curl / wget url       # 访问网页、下载文件
netstat -tulnp        # 查看本机监听端口
ss -tulnp             # 新版替代netstat，更快

# 指定输出文件-o
gcc main.c -o app
arm-none-eabi-gcc main.c -o main.elf

# 指定头文件目录-I
gcc main.c -I./inc -o app

# 指定库路径+链接库 -L -l
gcc main.c -L./lib -luart -o app

# 开启调试信息-g
gcc main.c -g -o app

# 代码优化等级 O0/O1/O2/O3
gcc main.c -O2 -o app

# 自定义宏定义-D
gcc main.c -DDEBUG -o app

# 开启全部警告Wall
gcc main.c -Wall -o app

# 查看gcc帮助文档
gcc -h
gcc --help
man gcc
